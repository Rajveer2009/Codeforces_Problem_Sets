## Compiling and Running a Solution

1. **Navigate to the problem's directory**  
   Example:
   ```
   cd "1030A - In Search of an Easy Problem"
   ```

2. **Compile the code**  
   ```
   g++ main.cpp -o solution
   ```

3. **Run the executable**  
   ```
   ./solution
   ```
